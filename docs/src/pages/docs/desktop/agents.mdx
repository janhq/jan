# Agents

Jan supports autonomous AI agents that run entirely on your own hardware — no cloud required.

<Callout type="warning">
The OpenClaw agent integration was removed in **Jan v0.7.9** due to resource usage and security concerns in local AI environments. Agent support is being redesigned for a simpler, safer experience in a future release.
</Callout>

## Launch Agents via CLI

You can run agents directly from your terminal using the Jan CLI. The `jan launch` command starts a local model and wires it to an agent automatically — no manual configuration needed.

```bash
jan launch claude                                       # pick a model, then run Claude Code against it
jan launch claude --model janhq/Jan-code-4b-gguf        # use a specific model with Claude Code
jan launch openclaw --model janhq/Jan-code-4b-gguf      # wire openclaw to a local model
```

See the [Jan CLI](/docs/desktop/cli) documentation for full options.
