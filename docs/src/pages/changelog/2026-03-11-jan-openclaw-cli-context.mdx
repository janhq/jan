# Highlights

**OpenClaw Integration**
- Integrate OpenClaw with a single click — run it locally with your favorite model, fully private
- Safe, sandboxed environment setup via Docker
- New Agent Chat in Jan — ask your agent to work on tasks directly from the app
- Connect to Telegram and WhatsApp through a simple setup

**Jan CLI**
- Run Jan headless from the terminal — no desktop app required
- Serve models and launch your favorite agents like Claude Code and OpenClaw from the command line

**Automatic Context Management**
- Jan now automatically manages context size so you never get cut off mid-conversation
- Context window is dynamically adjusted to keep conversations flowing without manual tuning
