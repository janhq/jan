# Highlights

**Simplified Onboarding Flow**
- Completely redesigned first-time user experience for faster setup
- Quick start with Jan's recommended model - get chatting in seconds
- Streamlined extension installation with clear instructions
- Background model preparation so you can explore while Jan gets ready

**Enhanced File Attachments**
- Desktop file attachments now fully enabled for richer conversations
- Improved attachment handling with graceful error recovery
- Re-submit attachments seamlessly when needed
- Better file parsing for more accurate context extraction

**Smarter Browser Integration**
- Jan Browser MCP now available as a chat input shortcut for instant web access
- New dialog to easily install the Jan Browser extension
- Updated link to Chrome Web Store for quick extension setup
- Proactive mode moved to MCP settings for better organization

**CUDA 13 Support**
- Added CUDA 13 backend support for latest NVIDIA GPUs
- CUDA 13 backend prioritized for improved performance on compatible hardware

**Stability & Performance**
- Fixed MCP server memory allocation errors with JIT compilation
- Improved cleanup functions to ensure proper app exit
