# Highlights

**Native MLX Support on macOS**
- Adds MLX as an accelerated backend option for Apple silicon
- Expands backend support for local inference on macOS

**App-wide UX and UI Refresh**
- Refreshed UX and UI across the app, beyond the chat screen
- Improved layout, navigation, and consistency across key surfaces

**Projects: File Upload Support**
- Upload files directly into Projects
- Work with documents and files alongside your project chats

**Local API Server Improvements**
- Local API server now supports both local and remote models
- More flexible setup for integrations and developer workflows

**Improved CLI Compatibility**
- Better compatibility with Claude Code and other CLI tools
- Smoother local workflows when connecting Jan to external tools
