## Default Permission

Default permissions for the vector-db plugin

#### This default permission set includes the following:

- `allow-get-status`
- `allow-create-collection`
- `allow-insert-chunks`
- `allow-create-file`
- `allow-search-collection`
- `allow-delete-chunks`
- `allow-delete-file`
- `allow-delete-collection`
- `allow-chunk-text`
- `allow-list-attachments`
- `allow-get-chunks`
